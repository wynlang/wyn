## What Works Now

### Extension Methods (Two Syntaxes!)

**Syntax 1: Extension Method**
```wyn
struct Point { x: int, y: int }

fn Point.sum(p: Point) -> int {
    return p.x + p.y;
}
```

**Syntax 2: Impl Block**
```wyn
struct Point { x: int, y: int }

impl Point {
    fn sum(self) -> int {
        return self.x + self.y;
    }
}
```

**Both work identically:**
```wyn
fn main() -> int {
    let p = Point { x: 3, y: 4 };
    return p.sum();  // Returns 7
}
```
