## What Cannot Be Implemented (Requires Language Changes)

### 1. Array Functional Methods
**Requires:** First-class functions, closures, lambda expressions

**Missing:**
- `.map(fn)` - Transform elements
- `.filter(fn)` - Filter elements  
- `.reduce(fn, init)` - Reduce to value
- `.find(fn)` - Find first match
- `.any(fn)` / `.all(fn)` - Predicate checks

**Why:** Wyn doesn't have function types or closures yet.

### 2. Networking
**Requires:** Async/await, threading, event loop

**Missing:**
- TCP/UDP sockets
- HTTP client/server
- WebSocket support

**Why:** Needs runtime support for async operations.

### 3. Regex
**Requires:** Complex parser or external library integration

**Missing:**
- Pattern compilation
- Match/search operations
- Capture groups

**Why:** Would need full regex engine implementation.
